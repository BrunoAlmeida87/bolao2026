📖 Resumo das Regras

  • Placar Exato → maior pontuação
  • Vencedor + Saldo de Gols → pontuação intermediária
  • Apenas o Vencedor → pontuação básica
  • Palpite de Campeão → bônus especial ao acertar o campeão
  • Palpites são bloqueados pelo admin pouco antes de cada jogo
  • Desempate: exatos → vencedor+saldo → vencedor
